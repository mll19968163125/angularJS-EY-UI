** 文件目录结构 ** 

    app/  
        home/                      -->登录模块文件夹
            controllers/            -->控制器文件夹
            directives/             -->指令文件夹
            services/               -->服务文件夹 
        login/                      -->登录模块文件夹
            controllers/            -->控制器文件夹
            directives/             -->指令文件夹
            services/               -->服务文件夹   
    framework/                      -->主程序入口
    theme/
        css/                        -->项目的样式文件夹
        img/                        -->项目的图片文件夹    
    lib/                            -->项目使用的第三方库以及自定义库
    index.html                      -->应用的入口文件
    main.js                         -->加载语言等信息
